# AI Email Assistant — Email-Native Operational Intelligence

AI Email Assistant turns your inbox into an operations layer: it reads incoming email, understands what it's *asking for*, and turns it into action — analysis, structured output, task assignment, and approval launches — without you leaving email.

This repo is the **product site + interactive demo** that walks through the concept: how an email becomes an AI analysis, a structured output, and a triggered action.

## What it shows

- **How it works** — email in → AI analysis → structured output → action triggered
- **Live demo steps** — a walkthrough of AI Email Assistant processing a request end to end
- **About / use cases** — assigning tasks and launching approvals straight from email
- **Coming soon** — what's next on the roadmap

## Tech stack

- [Vite](https://vitejs.dev/) + [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- [lucide-react](https://lucide.dev/) icons

## Run it

```bash
npm install
npm run dev      # dev server
npm run build    # production build
```
